# Claude AI Prompts Collection

A comprehensive collection of AI agent prompts and design system workflows for Claude AI.

## Repository Structure

### `/claude-agents/`
Professional AI agent prompts for software development teams:

- **Product-Manager.md** - Transform raw ideas into structured product plans with user stories and feature backlogs
- **UX-UI-Designer.md** - Design user experiences with comprehensive style guides and user research
- **System-Architect.md** - Create scalable system architectures with technical specifications
- **Senior-Backend-Engineer.md** - Build robust backend systems with best practices
- **Senior-Frontend-Engineer.md** - Develop modern frontend applications
- **QA-Test-Automation-Engineer.md** - Design comprehensive testing strategies
- **DevOps-Deployment-Engineer.md** - Manage CI/CD pipelines and infrastructure
- **Security-Analyst.md** - Conduct security assessments and vulnerability analysis

### `/design-system/`
Design system creation and implementation workflows:

- **01-Extract-Opinions.md** - Extract style guides from UI screenshots and design references
- **02-Merge-Ideas-Together.md** - Combine multiple style guides into a cohesive design system
- **03-Build-Prompt.md** - Generate functional UI designs based on style guides
- **Design-It.md** - SaaS product design template with complete guidelines
- **Expand-It.md** - Add philosophy and usage guidelines to style guides
- **Extract-It.md** - UX/UI style guide creation framework
- **Merge-It.md** - Merge reference design systems into new forms

## Usage

### Claude Agents
Each agent prompt is designed to be used independently with Claude AI. Simply:

1. Copy the content of the desired agent prompt
2. Paste it into your Claude conversation
3. Provide your specific requirements or context
4. Claude will respond following the agent's specialized methodology

### Design System Workflows
The design system prompts follow a sequential workflow:

1. **Extract** - Analyze competitor designs and extract style patterns
2. **Expand** - Add philosophy and implementation guidelines
3. **Merge** - Combine insights into a unified design system
4. **Build** - Generate functional UI designs from the system
